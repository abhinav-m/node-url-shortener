Belittle - A simple url shortener api.

Example Usage:

https://belittle.herokuapp.com//https://www.github.com Go!

https://belittle.herokuapp.com/new/hghps://wss.github.com Go!

Example Output:

{url: "https://www.github.com", short_url: "https:/belittle.herokuapp.com/5" }

{ error: "Please submit a valid url." }
